Tipsy To-do list requirements:

Users can:
    create new account
    create new to-dos
    delete to-dos
    mark importance
    set completion date
    mark to-dos as done
    make notes about to-dos
    set alerts

Users:
    id
    email account
    password
    name
Tasks:
    id
    title
    created_at
    completed_at

User / Task relationships:
    user can have many tasks
    task belongs to one user
